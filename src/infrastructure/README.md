# Infrastructure

Cross-cutting concerns: Polly policies, DI setup, logging, and configuration.

## Purpose
This project will contain:
1. Polly resilience policies (retry, circuit breaker, timeout)
2. Dependency injection configuration
3. Logging setup and structured log contracts
4. Configuration management (Azure Key Vault, app settings)
5. Health checks and diagnostics

## Key Dependencies (planned)
- Polly
- Microsoft.Extensions.DependencyInjection
- Microsoft.Extensions.Logging
- Azure.Identity
- Azure.Security.KeyVault.Secrets
