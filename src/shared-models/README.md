# Shared Models

DTOs, AutoMapper profiles, and shared contracts used across the pipeline.

## Purpose
This project will contain:
1. Shared DTOs for rule workflows, rules, metadata, and traceability
2. AutoMapper profiles for mapping between pipeline stages
3. JSON serialization contracts
4. Common constants and enums

## Key Dependencies (planned)
- AutoMapper
- System.Text.Json
- RulesEngine (Microsoft) — for workflow/rule model alignment
